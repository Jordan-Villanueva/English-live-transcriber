# 🎤 English Live Transcriber

Aplicación web para transcribir audio en **tiempo real** y mostrar traducción bilingüe 
(inglés ↔ español), pensada como herramienta didáctica para clases.

## 🚀 Características
- Captura voz desde el micrófono.
- Transcripción en inglés con Whisper.
- Traducción automática al español.
- Visualización en tiempo real en navegador.
